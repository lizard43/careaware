Help

  • General
    • Use the blue menu bar to switch between pages.
    • Change the Save folders on the Preferences Page before using for the FIRST time.
    • Close browser window to exit application.
  • Radiology Report
    • Select Radiology Report on the menu bar.
    • Before using the page for the FIRST time, use the Preferences Page to setup.
    • Enter Patient's information and click Save.
    • Info is saved to a Word file in folder specified on the Preferences Page. 
    • The Word filename is built using the current date.  You can modify the filename at the bottom of the Radiology Report Page.
